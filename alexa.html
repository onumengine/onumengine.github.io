Projects

Here are some of the projects I've built:

Medizorg

Medizorg is an application that leverages the power of AI chatbots to provide a user with a provisional diagnosis and uses video conferencing for telemedicine.

It is being built with Flutter's BLoC architecural pattern and has features like Video Calling, User Authentication, Device Permission mnagement and internet connectivity.

Musicplayer

MusicPlayer is a Flutter application that reaches deep into Android platform APIs through method channels, an amazing feature of Flutter, to read and play music files stored on any android device.

It can modify files on a device and store data locally.

KTNotes

KTNotes is a psychology experiment. It's a notekeeper app built with a unique color combination and to support emojis and fancy text.

These features were integrated to test how much a simple app can engage a user if its user experience is impressive.

It's a native android application built first with Java and then refactored to Kotlin. It implements recyclerView, stores notes in a local SQLite database and is colorful and fun to use.
